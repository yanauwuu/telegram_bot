# Telegram Bot with AI Assistant

Учебный проект по разработке Telegram-бота на Python.

Изначально в рамках лабораторной работы требовалось реализовать обычного Telegram-бота с базовыми командами и обработкой сообщений. Помимо основной функциональности, я расширила проект и добавила модуль AI-ассистента, который умеет отвечать на вопросы по текстовой базе знаний.

## Основная функциональность

Бот поддерживает базовые команды:

- `/start` — приветствие и описание возможностей;
- `/quote` — получение цитаты дня;
- `/headlines` — получение новостных заголовков;
- `/weather` — получение погоды по введённому городу;
- `/clear` — очистка истории диалога с AI-ассистентом.

Также бот умеет обрабатывать обычные текстовые сообщения пользователя.

## AI Assistant

Дополнительно был реализован простой AI-ассистент с RAG-подходом.

Пользователь может задать вопрос, после чего бот:

1. ищет релевантные фрагменты в текстовой базе знаний;
2. формирует контекст для ответа;
3. собирает prompt для LLM;
4. при наличии подключённого LLM API отправляет запрос в модель;
5. возвращает пользователю ответ.

Если LLM API не подключён, бот всё равно работает: он показывает найденный контекст из базы знаний. Это позволяет тестировать логику поиска и обработки текста без внешней модели.

## Работа с файлами

Бот может принимать `.txt`-файлы от пользователя. После загрузки файла ассистент использует его как новую базу знаний и отвечает на вопросы уже по содержимому этого документа.

## Что было добавлено сверх требований лабораторной

Помимо обычной логики Telegram-бота, были добавлены:

- разбиение текста на фрагменты;
- поиск релевантного контекста по вопросу пользователя;
- формирование prompt для LLM;
- интеграция с LLM API;
- fallback-режим без подключённой LLM;
- загрузка пользовательских `.txt`-файлов;
- хранение истории диалога;
- отдельная структура модулей для AI-ассистента.
