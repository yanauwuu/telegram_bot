# telegram-bot

Телеграм-бот на Python с long polling, асинхронной обработкой запросов и веб-скрапингом.

## Описание
Проект представляет собой консольного Telegram-бота, который работает с Telegram Bot API, получает сообщения через long polling и обрабатывает команды пользователей.  
В боте реализованы синхронная и асинхронная версии, а также функции веб-скрапинга и простого управления состоянием пользователей.

## Возможности
- получение и обработка обновлений
- отправка сообщений
- эхо-бот
- асинхронные HTTP-запросы
- веб-скрапинг
- конкурентное выполнение задач
- многошаговые диалоги

## Технологии
- Python
- asyncio
- aiohttp
- BeautifulSoup4
